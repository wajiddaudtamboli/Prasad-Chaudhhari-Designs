# Prasad Choudhhari Designs - Luxury Interior Portfolio

A modern, responsive portfolio website for Prasad Choudhhari Designs, showcasing premium residential interior design services. Built with React, Vite, and Tailwind CSS.

## 🌐 Live Demo

The website is deployed on Vercel and ready for production use.

## ✨ Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop views
- **Smooth Animations**: Framer Motion animations for engaging user interactions
- **Cost Calculator**: Interactive estimate calculator for home interior projects
- **Contact Forms**: Multiple contact methods (phone, WhatsApp, email)
- **Modern UI/UX**: Luxury-focused design with premium color palette
- **Performance Optimized**: Fast loading with Vite build optimization
- **All Buttons Working**: Fully tested interactive elements

## 🛠️ Tech Stack

- **Frontend Framework**: React 18.3
- **Build Tool**: Vite 5.4
- **CSS Framework**: Tailwind CSS 3.4
- **Animations**: Framer Motion 11.11
- **Icons**: Lucide React 0.468

## 🚀 Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
git clone https://github.com/wajiddaudtamboli/Prasad-Chaudhhari-Designs.git
cd Prasad-Chaudhhari-Designs
npm install
npm run dev
```

## 📋 Available Commands

```bash
npm run dev      # Start development server (port 5174)
npm run build    # Build for production
npm run preview  # Preview production build locally
```

## 🚀 Deploy on Vercel

### Option 1: GitHub Integration
1. Push code to GitHub
2. Connect repo to Vercel at https://vercel.com
3. Automatic deployment on push

### Option 2: Vercel CLI
```bash
npm install -g vercel
vercel
```

### Option 3: Manual Deployment
```bash
npm run build
# Upload dist/ folder to hosting service
```
