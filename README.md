# Prasad Choudhhari Designs Website

Production-ready one-page luxury residential interior design studio website built with React, Vite, Tailwind CSS, Framer Motion, and Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

Import this folder into Vercel. The default settings work:

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Notes

- Contact details are dummy placeholders from the brief.
- Interior images are remote Unsplash placeholders.
- The logo asset is stored in `public/logo.svg`.
